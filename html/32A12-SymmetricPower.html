symmetric power


Let X be a set and let

Xm:=X×⋯×X⏟m-times.

Denote an element of Xm by x=(x1,…,xm). Define an equivalence relation by x∼x′ if and only if there exists a permutation σ of (1,…,m), such that xi=xσ⁢i′.

Definition.

The mth symmetric power of X is the set Xs⁢y⁢mm:=Xm/∼. That is, the set of equivalence classes of Xm under the relation ∼.

Let π be the natural projection of Xm onto Xs⁢y⁢mm.

Proposition.

f:Xm→Y is a symmetric function if and only if there exists a function g:Xs⁢y⁢mm→Y such that f=g∘π.

From now on let R be an integral domain. Let τ′:Xm→Xm be the map τ′⁢(x):=(τ1⁢(x),…,τm⁢(x)), where τk is the kth elementary symmetric polynomial. By the above lemma, we have a function τ:Xs⁢y⁢mm→Xm, where τ′=τ∘π.

Proposition.

τ is one to one. If R is algebraically closed, then τ is onto.

A very useful case is when R=ℂ. In this case, when we put on the natural complex manifold structure onto ℂs⁢y⁢mm, the map τ is a biholomorphism of ℂs⁢y⁢mm and ℂm.
